# 실시간 채팅

socket.io 라이브러리를 이용한 실시간 채팅 서비스 입니다.


# 개발환경

Framework: Next.js

Database: MongoDB



# 주요기능

* 방 채팅
  - 닉네임 설정(중복불가)
  - 방 이름 설정(중복가능)
  - 비밀번호 방
  - 새로고침
* 랜덤 채팅
  - 1:1 채팅
* AI채팅
  - AI와 1:1 채팅
  - OpenAI의 ChatGPT API활용
